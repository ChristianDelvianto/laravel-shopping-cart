# Simple E-Commerce Shopping Cart

A small Laravel-based e-commerce shopping cart application built as a practical task.  
The project focuses on clean architecture, Laravel best practices, and backend fundamentals rather than UI complexity.

---

## Tech Stack

- **Backend:** Laravel
- **Frontend:** Vue 3 + Inertia.js
- **Authentication:** Laravel Breeze (session-based)
- **Styling:** Tailwind CSS
- **Database:** MySQL / PostgreSQL
- **Queue:** Laravel Jobs (database driver)
- **Scheduler:** Laravel Task Scheduling
